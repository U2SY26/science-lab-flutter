# Google Play Store 등록정보

## 앱 이름 (30자 이내)
눈으로 보는 과학

## 짧은 설명 (80자 이내)
물리와 수학을 시뮬레이션으로 직접 보고 이해하세요. 공식 암기 없이 원리를 깨우칩니다.

## 전체 설명 (4000자 이내)
물리와 수학, 어렵게만 느껴지셨나요?

'눈으로 보는 과학'은 복잡한 공식과 이론을 직관적인 시뮬레이션으로 보여주는 과학 학습 앱입니다.

### 주요 기능

- 실시간 물리 시뮬레이션
  운동, 힘, 에너지 등 물리 법칙을 직접 눈으로 확인하세요.

- 수학 시각화
  함수, 그래프, 벡터 등 추상적인 개념을 시각적으로 이해할 수 있습니다.

- 인터랙티브 실험
  변수를 조절하며 결과가 어떻게 달라지는지 직접 실험해보세요.

- 단계별 학습
  기초부터 심화까지 체계적으로 학습할 수 있습니다.

### 이런 분께 추천합니다

- 물리/수학 개념이 잘 이해되지 않는 학생
- 시각적 학습을 선호하는 분
- 과학에 흥미를 갖고 싶은 분
- 자녀의 과학 교육에 관심 있는 학부모

공식을 외우는 것이 아닌, 원리를 이해하는 진짜 과학 학습을 시작하세요!

## 필요한 그래픽 에셋

### 1. 앱 아이콘
- 512 x 512 px (PNG, 32비트)
- 위치: assets/icon/app_icon.png (이미 생성됨)

### 2. 기능 그래픽 (Feature Graphic)
- 1024 x 500 px
- Play Store 상단에 표시되는 배너 이미지

### 3. 스크린샷
- 최소 2장, 최대 8장
- 권장: 16:9 또는 9:16 비율
- 실제 앱 화면 캡처

### 4. 개인정보처리방침 URL
- Play Store 등록 시 필수
- 웹페이지로 호스팅 필요
