# Google Play Store 등록 자료

## 앱 이름
눈으로 보는 과학 - 3DWeb Science Lab

## 짧은 설명 (80자 이내)
물리, 수학, AI/ML 시뮬레이션으로 과학 원리를 직접 체험하세요!

## 전체 설명 (4000자 이내)
**손끝으로 느끼는 우주의 법칙!**

눈으로 보는 과학은 복잡한 과학과 수학의 원리를 인터랙티브 시뮬레이션으로 쉽게 이해할 수 있도록 도와주는 교육용 앱입니다.

📚 **62개 이상의 시뮬레이션**
• 물리학: 단진자, 이중 진자, 발사체 운동, 스프링 진동, 전기장
• 수학: 수식 그래프, 이차함수, 벡터, 푸리에 변환, 만델브로트 집합
• 혼돈 이론: 로렌츠 어트랙터, 3체 문제, 로지스틱 맵
• AI/ML: 신경망, 경사하강법, K-Means, SVM, 결정 트리

🎮 **직접 조작하며 배우기**
• 파라미터를 실시간으로 조절하며 변화 관찰
• 다양한 초기 조건과 프리셋 제공
• 수식과 함께 원리 설명

🌟 **주요 특징**
• 아름다운 다크 테마 UI
• 카테고리별 필터링 및 검색
• 학습 진행률 추적
• 즐겨찾기 기능
• 오프라인 사용 가능

🔄 **지속적인 업데이트**
새로운 시뮬레이션과 기능이 계속 추가됩니다!

🌐 **웹 버전**
https://3dweb-rust.vercel.app

중학생부터 대학생, 그리고 과학에 관심 있는 모든 분들께 추천합니다.
