# 스토어 등록 자료 가이드

## 스크린샷 촬영 가이드

### 필요한 스크린샷 (5~8장)

1. **홈 화면** - 시뮬레이션 목록 전체 보기
2. **단진자 시뮬레이션** - 물리 카테고리 대표
3. **만델브로트 집합** - 수학 프랙탈 대표
4. **로렌츠 어트랙터** - 혼돈 이론 대표
5. **신경망 플레이그라운드** - AI/ML 대표
6. **푸리에 변환** - 인터랙티브 시각화
7. **K-Means 클러스터링** - 머신러닝 대표
8. **전기장 시각화** - 물리 인터랙티브

### 스크린샷 해상도

| 스토어 | 해상도 |
|-------|-------|
| Play Store | 1080x1920 (Phone), 1200x1920 (7" Tablet) |
| App Store | 1290x2796 (6.7"), 1242x2688 (6.5"), 1179x2556 (6.1") |

### 촬영 방법

```bash
# Android 에뮬레이터에서 스크린샷
flutter run
# 에뮬레이터에서 Ctrl+S 또는 카메라 버튼

# 또는 adb 사용
adb exec-out screencap -p > screenshot.png
```

## 스토어 설명문

### 짧은 설명 (80자)
```
물리, 수학, AI를 눈으로 보고 손으로 만지며 이해하는 과학 시뮬레이터
```

### 긴 설명
```
🔬 눈으로 보는 과학 - 어려운 과학을 쉽게!

복잡한 공식과 이론을 인터랙티브 시뮬레이션으로 직접 체험해보세요.

📚 30개 이상의 시뮬레이션
• 물리: 단진자, 이중진자, 파동간섭, 전기장, 발사체 운동
• 수학: 푸리에 변환, 만델브로트 집합, 그래프 이론, 벡터
• 혼돈 이론: 로렌츠 어트랙터, 3체 문제, 로지스틱 맵
• AI/ML: 신경망, 경사하강법, K-Means, SVM, PCA, 결정트리

✨ 특징
• 터치로 조작하는 실시간 시뮬레이션
• 파라미터를 조절하며 결과 확인
• 아름다운 다크 테마 디자인
• 오프라인에서도 사용 가능

🎓 이런 분께 추천합니다
• 물리/수학을 공부하는 학생
• AI/머신러닝에 관심있는 분
• 과학을 시각적으로 이해하고 싶은 분
• 자녀에게 과학을 쉽게 설명하고 싶은 부모님

지금 바로 우주의 법칙을 손끝으로 느껴보세요!
```

## 개인정보처리방침 URL

`privacy_policy.html` 파일을 다음 방법으로 호스팅:

1. **GitHub Pages** (무료)
   - 레포지토리에 파일 업로드
   - Settings > Pages에서 활성화
   - URL: `https://username.github.io/repo/privacy_policy.html`

2. **Firebase Hosting** (무료)
   ```bash
   firebase init hosting
   firebase deploy
   ```

3. **Netlify** (무료)
   - 드래그 앤 드롭으로 배포
